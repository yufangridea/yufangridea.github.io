Gridea 使用远程 Github Pages 搭建静态博客

2021-12-10 1 min read # Gridea

👀上次的 SFTP 需要自己有台服务器才行。

😬这次来白嫖 Github Pages 搭建 gridea

😅Github 在国内访问有时候不太好，必要时需要🪜。

1. 创建一个Personal access tokens

打开GitHub，登录 GitHub

点击右上角 settings ，下拉在左侧找到 Developer settings

选择 Personal access tokens ，点击 Generate new token 的按钮，按照下图设置：

💥💥💥
复制好 Personal access tokens

2. 创建仓库存储静态网页

右上角 + 号， New repository，按照下图要点设置

进到仓库，setting-pages，绑定自己的域名，并设置 cname 记录

💥💥💥
Github 需要10 分钟 左右才能自动配置好证书！

在 Gridea 填写域名，仓库名称等信息

点击同步，访问你的域名，应该可以成功访问了！🎉
