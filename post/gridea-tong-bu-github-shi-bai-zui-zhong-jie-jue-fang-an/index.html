Gridea 同步 Github 失败最终解决方案

2021-12-10 1 min read # Gridea

Github 不仅有时候网页打不开，同步也不行💀

这个解决方案来自 这里，感谢这位大佬👍

你必须要有一个 Socks5，列如下面的：

下载proxifier ，这个工具可以指定某个软件按照你的规则走

Mac版本地址：https://www.macwk.com/soft/proxifier
Window版本地址：https://www.hanzify.org/software/13717.html

安装打开，顶部配置文件-xx服务器-添加

配置文件-xx规则-添加，选择 Gridea.exe 的位置，到最后应该如下图：

配置文件-名称解析-通过xx解析主机名称

此时打开 Gridea，测试下~🎉
