Virmach DD 纯净系统

2021-12-18 2 min read # DD # Virmach

建议先通过 VNC 连接到小鸡，不然后面可能部分过程看不到，不过也可以通过 Ping 来看是否安装完成。

网页后台无法 VNC 的，下载 TurboVNC-2.2.6-x64.exe 来连接。
下载地址：
https://wws.lanzouo.com/iMq05xqb8dg

dd 脚本来自：这里

在 Virmach 的 Debian8 系统测试通过！

更新：

#Debian/Ubuntu:
apt-get update
#RedHat/CentOS:
yum update

安装必要软件：

#Debian/Ubuntu:
apt-get install -y xz-utils openssl gawk file
#RedHat/CentOS:
yum install -y xz openssl gawk file

DD debian 10示例：

wget -N --no-check-certificate https://raw.githubusercontent.com/veip007/dd/master/InstallNET.sh && chmod +x InstallNET.sh && ./InstallNET.sh -d 10 -v 64 -p "自定义root密码" -port "自定义ssh端口"

等待一会。

其他脚本：

bash <(wget --no-check-certificate -qO- 'https://raw.githubusercontent.com/MoeClub/Note/master/InstallNET.sh') -d 10 -v 64 -p 密码 -port ssh端口

低内存需要进 VNC 选择两个选项。
